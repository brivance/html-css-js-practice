Links

This example is about links and images.

This particular page is about links! There are three kinds of links:

  • Absolute links, like to Mozilla Developer Network, which is a very good resource for webdevelopers. It gives links to URLs outside our website.
  • Relative links, like to our extras page. This is relative to the page (aka file) you are currently in.
  • Root-relative links, like to the home page of our website. This is useful to get a link starting from the root (home page) path.
